Buffers management in glium

There are three levels of abstraction in glium:

  • A Buffer corresponds to an OpenGL buffer object. This type is not public.
  • A SubBuffer corresponds to a part of a Buffer. One buffer can contain one or multiple subbuffers.
  • The VertexBuffer, IndexBuffer, UniformBuffer, PixelBuffer, ... types are abstractions over a subbuffer indicating their specific purpose. They implement Deref for the subbuffer. These types are in the vertex, index, ... modules.
